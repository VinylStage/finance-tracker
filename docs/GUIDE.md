# 사용자 가이드

Finance Tracker 사용법을 기능별로 정리한 문서입니다. 앱 상단 네비게이션의 각 탭과 이름이 동일합니다.

## 대시보드

이번 달 수입/지출 현황과 잔액 추이를 한눈에 확인합니다. 카테고리별 지출 비중과 예산 대비 사용률을 함께 보여줍니다.

## 거래입력

수입·지출 거래를 직접 입력하거나 목록에서 검색·수정·삭제합니다. 체크박스로 여러 건을 선택해 일괄 삭제할 수 있고, Shift+클릭으로 범위 선택도 가능합니다.

카드사 엑셀 파일을 올려 대량으로 등록하려면 **설정 > 카드사 엑셀 임포트**를 이용하세요 (농협·롯데·삼성·하나·현대 지원). 같은 승인번호를 가진 거래는 중복으로 간주되어 자동으로 건너뜁니다.

## 기간비교

두 기간(예: 이번 달과 지난 달)의 수입·지출을 비교해 변화 추이를 파악합니다.

## 할부

할부 결제 건의 남은 회차와 월별 상환액을 관리합니다.

## 리볼빙

리볼빙 결제 잔액과 이자, 상환 계획을 추적합니다.

## 부채

대출 등 부채 항목의 잔액과 상환 스케줄을 관리합니다.

## 적금

적금 상품의 만기와 목표 금액, 현재 납입 현황을 관리합니다.

## 시뮬레이터

향후 수입/지출 변화를 가정해 잔액 추이를 미리 시뮬레이션합니다.

## 설정

- **기본 설정**: 초기 잔액, 월 수입 기준값
- **카테고리 / 결제수단 관리**: 추가·수정·비활성화
- **데이터 내보내기**: 거래내역 CSV, 전체 백업 JSON
- **설정 백업/복원**, **거래내역 백업/복원**
- **카드사 엑셀 임포트**
- **위험 구역**: 전체 거래내역 삭제 (되돌릴 수 없으니 백업 후 사용)
